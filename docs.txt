## `linkcovery-docs links`

Manage your bookmarked links

**Usage**:

```console
$ linkcovery-docs links [OPTIONS] COMMAND [ARGS]...
```

**Options**:

* `--help`: Show this message and exit.

**Commands**:

* `add`: Add a new link to your bookmarks.
* `list`: List your bookmarked links.
* `search`: Search your bookmarks.
* `show`: Show detailed information about a specific...
* `edit`: Edit an existing link.
* `delete`: Delete a link from your bookmarks.
* `mark-read`: Mark a link as read.
* `mark-unread`: Mark a link as unread.

### `linkcovery-docs links add`

Add a new link to your bookmarks.

**Usage**:

```console
$ linkcovery-docs links add [OPTIONS] URL
```

**Arguments**:

* `URL`: URL to bookmark  [required]

**Options**:

* `-d, --desc TEXT`: Description for the link
* `-t, --tag TEXT`: Tag to categorize the link
* `-r, --read`: Mark as already read
* `--no-fetch`: Skip fetching metadata from URL
* `--help`: Show this message and exit.

### `linkcovery-docs links list`

List your bookmarked links.

**Usage**:

```console
$ linkcovery-docs links list [OPTIONS]
```

**Options**:

* `-l, --limit INTEGER`: Maximum number of links to show  [default: 20]
* `--read-only`: Show only read links
* `--unread-only`: Show only unread links
* `--help`: Show this message and exit.

### `linkcovery-docs links search`

Search your bookmarks.

**Usage**:

```console
$ linkcovery-docs links search [OPTIONS] [QUERY]
```

**Arguments**:

* `[QUERY]`: Search in URLs, descriptions, and tags

**Options**:

* `--domain TEXT`: Filter by domain
* `-t, --tag TEXT`: Filter by tag
* `--read-only`: Show only read links
* `--unread-only`: Show only unread links
* `-l, --limit INTEGER`: Maximum results  [default: 20]
* `--help`: Show this message and exit.

### `linkcovery-docs links show`

Show detailed information about a specific link.

**Usage**:

```console
$ linkcovery-docs links show [OPTIONS] LINK_ID
```

**Arguments**:

* `LINK_ID`: Link ID to display  [required]

**Options**:

* `--help`: Show this message and exit.

### `linkcovery-docs links delete`

## `linkcovery-docs data`

Import and export your bookmark data

**Usage**:

```console
$ linkcovery-docs data [OPTIONS] COMMAND [ARGS]...
```

**Options**:

* `--help`: Show this message and exit.

**Commands**:

* `export`: Export all your links to a JSON file.
* `import`: Import links from a JSON file.

### `linkcovery-docs data export`

Export all your links to a JSON file.

**Usage**:

```console
$ linkcovery-docs data export [OPTIONS] [OUTPUT]
```

**Arguments**:

* `[OUTPUT]`: Output file path  [default: links.json]

**Options**:

* `-f, --force`: Overwrite existing file
* `--help`: Show this message and exit.

### `linkcovery-docs data import`

Import links from a JSON file.

**Usage**:

```console
$ linkcovery-docs data import [OPTIONS] INPUT_FILE
```

**Arguments**:

* `INPUT_FILE`: JSON file to import  [required]

**Options**:

* `--dry-run`: Show what would be imported without actually importing
* `--help`: Show this message and exit.
